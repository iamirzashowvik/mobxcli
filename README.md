# Usage
<!-- usage -->
```sh-session
$ npm install -g mvvmmobxcli
$ mobxcli COMMAND
running command...
$ mobxcli (--version)
mvvmmobxcli/1.0.2 darwin-arm64 node-v16.13.2
$ mobxcli --help [COMMAND]
USAGE
  $ mobxcli COMMAND
To generate your boilerplate code run
$ mobxcli resource your_page
...
```
